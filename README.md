## Run Locally

Clone the project

```bash
  git clone git@github.com:Tonips22/Nothing-Phone-2.git
```

Go to the project directory

```bash
  cd Nothing-Phone-2
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
